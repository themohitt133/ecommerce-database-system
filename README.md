# 🛒 E-Commerce Database System

## 📌 Project Overview

This is a portfolio-level PostgreSQL database project that simulates an e-commerce platform. The project demonstrates relational database design, SQL querying, business analysis, and reporting using realistic sample data.

## 🚀 Features

- User Management
- Product Management
- Order Management
- Payment Management
- Business Analysis Queries
- SQL Reporting

## 🗄️ Database Schema

The database contains four relational tables:

- Users
- Products
- Orders
- Payments

1. Open PostgreSQL / pgAdmin.
2. Create the database.
3. Execute `database.sql`.
4. Execute `queries.sql`.
5. View the query outputs.
